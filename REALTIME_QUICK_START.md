## Quick Troubleshooting

### Nothing happens?
- Make sure customer is on "Account → My Orders" page
- Then admin approves
- If still nothing → check browser console (F12) for errors

### Animation doesn't show?
- Hard refresh: Ctrl+Shift+R

### Notification doesn't appear?
- Check browser notification settings

### Works in one tab but not another?
- That's normal! Cross-tab takes 1-2 seconds
- Wait a moment and check again
