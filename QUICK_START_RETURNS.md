# ⚡ QUICK START - Flipkart-Style Returns System

## 🎯 What You Need to Know

Your order return/refund system is now **live** and works exactly like Flipkart!

---

## 🚀 CUSTOMER: How to Return an Item

### Step 1: Find Your Order
1. Login to ANCRIO
2. Go to **Account** → **Order History**
3. Find the delivered item you want to return
4. Look for "🔄 Return/Replace" button (only shows for delivered items within 7 days)

### Step 2: Open Return Form
- Click the **"🔄 Return/Replace"** button
- Modal opens with 3-step form

### Step 3: Fill Step 1 (Reason & Type)
```
Select Reason:
- ❌ Defective/Broken Item
- 📏 Wrong Size Received
- 📦 Wrong Item Delivered
- 🖼️ Item Not as Described
- ⚠️ Poor Quality
- 🎨 Color Mismatch
- 🤔 Changed My Mind
- 📝 Other Reason

Select Return Type:
- 💰 Refund (Money back)
- 🔄 Replace (Get another)
```

### Step 4: Upload Photos (Step 2)
```
Upload at least 1 photo. Upload 2+ for FASTER approval!

Two ways to add photos:
1. 📷 Take Photo (click to use camera)
2. 📂 Upload Files (select from device)

Photos appear in preview grid below.
You can remove any photo by clicking the X button.

Timeline:
- 1 photo = 48-hour approval
- 2+ photos = 24-hour approval ⭐ FASTER!
```

### Step 5: Add Details (Step 3)
```
Optional: Write a message explaining the issue.
Example: "The stitching is broken after first wash"

Review the return policy info displayed.
```

### Step 6: Submit
- Click **"Submit Return Request"**
- See success message with refund amount & timeline
- ✅ You're done!

### Step 7: Track Status
- Return appears in order history
- Shows refund amount
- Shows approval timeline
- Shows verification status
- Updates when admin approves/rejects

## 👨‍💼 ADMIN: How to Process Returns

### Access Admin Panel
1. Login as admin
2. Click **Admin Panel** button
3. Go to **"Return/Replace"** tab

### Review Return Requests
You'll see **Return Request Cards** with:
- Customer name & email
- Product being returned
- Return reason & type
- 📸 Photos (click thumbnails to zoom)
- 💰 Refund amount
- 🎯 AI Risk Score (Green/Orange/Red)
- ✓ Verification Status
- 💬 Customer message
- 🚀 Action buttons

### Check Photos
- See all uploaded photos as thumbnails
- Click any photo to expand full-size
- Close photo to go back
- Photo number shown for reference

### Make Decision
Click one button:

**✅ APPROVE**
- Approves the return
- Status becomes "Approved"
- Customer can now ship item
- Refund pending receipt

**❌ REJECT**
- Denies the return
- Status becomes "Rejected"
- No refund processed
- Customer gets notification

**🚚 SCHEDULE PICKUP**
- Auto-approves the return
- Sets pickup date
- Arranges for customer shipment
- Status becomes "Picked"

## 🎯 Admin Dashboard Quick Reference

### Color Meanings
```
🟢 Green   = Approved / Verified
🟡 Orange  = Pending / Awaiting
🔴 Red     = Rejected / High Risk
🟣 Purple  = Secondary Info
```

### AI Risk Score
```
🟢 Low (0-35%)      → Safe to approve
🟡 Medium (36-65%)  → Review before deciding
🔴 High (66-100%)   → Likely fraudulent
```

### Verification Status
```
✅ Verified  = Clear evidence, 2+ photos
⏳ Pending   = Awaiting review
❌ Rejected  = Fraud detected or policy violation
```

## ❓ FAQ

**Q: How long to get refund?**
A: 7-10 days after we receive your return item.

**Q: Do I need 2 photos?**
A: No, minimum 1. But 2+ photos = 24-hour approval (faster)!

**Q: Can I take photos with my phone camera?**
A: Yes! Click "📷 Take Photo" to use your device camera.

**Q: What if I don't have photos?**
A: You can't submit without at least 1 photo.

**Q: When does admin review my return?**
A: Within 24-48 hours depending on photo count.

**Q: What if return is rejected?**
A: No refund processed. You keep the item.

**Q: Can I return after 7 days?**
A: No, returns only allowed within 7 days of delivery.

## 🔧 Troubleshooting

### Camera Not Working?
- Grant permission when browser asks
- Try a different browser
- Try file upload instead

### Photos Not Uploading?
- Check file size (should be reasonable)
- Try a different format (JPG, PNG)
- Refresh page and try again

### Can't See Return Button?
- Order must be "Delivered" status
- Must be within 7 days of delivery
- Item must be eligible for returns

### Admin Not Seeing Request?
- Refresh admin page
- Check "Return/Replace" tab (not other tabs)
- Look for "Pending" status
