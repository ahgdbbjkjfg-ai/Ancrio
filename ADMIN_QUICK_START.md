## 📋 Main Functions

| Feature | Location | What It Does |
|---------|----------|--------------|
| **Add Products** | 📦 Products → + Button | Create new items to sell |
| **Edit Products** | 📦 Products → Edit | Modify name, price, stock |
| **Delete Products** | 📦 Products → Delete | Remove items from store |
| **Add Offers** | 🎁 Offers → + Button | Create discounts/promotions |
| **View Orders** | 📋 Orders → Table | See all customer purchases |
| **Settings** | ⚙️ Settings | Configure store name & shipping |

## 🆘 Troubleshooting

**"Admin panel not opening?"**
→ Click ⚙️ button, enter password

**"Can't find password?"**
→ Default: `admin123`

**"Data disappeared?"**
→ Check browser cache, may need to refresh

**"Product not showing in store?"**
→ Refresh page (F5) or wait a moment
